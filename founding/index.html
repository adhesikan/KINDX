What is KINDX?

KINDX is a community-driven utility token on the Base blockchain designed to enable tipping, participation, recognition, and social coordination.

Founding phase

KINDX is launching as a fair, community-seeded token. Instead of insiders, early supporters can help bootstrap liquidity and become the first real users.

Program purpose

This phase helps fund on-chain liquidity, kindx.pro tooling, and community utilities (tipping, rewards, dashboards).
